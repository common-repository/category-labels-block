A plugin that add native blocks which allows you to easily display associated category/taxonomy on a WordPress post.

== Description ==

Category Labels Block provides a customizable way to display categories associated with the current post. Allowing you to enhance your posts with dynamic category labels and taxonomies.

== Installation ==

1. Download the plugin ZIP file.
2. Go to `WordPress Dashboard > Plugins > Add New`.
3. Upload the ZIP file and activate the plugin.
4. Navigate to any page or post edit screen, and add the **Category Labels Block**.

== Features ==

- **Scope to current post:** A control that allows you to only display categories related to your current WordPress post.
- **SEO Plugins Support:** This plugin supports displaying the primary post category via "Yoast" and "Rank Math" SEO plugin.
- **Taxonomy Support (PRO):** Ability to display custom post taxonomies.
- **Limit Categories/Taxonomies (PRO):** Ability to limit the total number of categories to display.

== Usage ==

1. Add the "Category Labels" block to start grouping categories or taxonomies.
2. Inside the "Category Labels" block, add a "Post Term" block to select specific categories or taxonomies.
3. Use the "Term Name" and "Term Abbreviation" blocks to display the category name and abbreviation, respectively.
4. Adjust visibility and limits using the block's control settings in the sidebar.

**Note:** The free version allows selection of post categories only. Premium version includes additional features like custom taxonomy selection and limit controls.

== Frequently Asked Questions ==

= Can I use this with any theme?

Yes, Category Labels Block works with any theme that supports Gutenberg blocks.

= Is there a limit to the number of categories I can display?

In the free version, you can display all categories related to the post. The premium version offers controls to limit the number of categories.
